# ADR-0002: 백테스트 통계는 비중복 표본과 순위상관으로 계산한다

- 상태: 채택됨
- 날짜: 2026-07-15
- 관련 코드: `src/analysis/backtest.py`의 `run_backtest()`, `_non_overlapping_values()`

## 맥락

백테스트 탭은 "우리 도구가 '저평가'라고 판정했던 과거의 날에 실제로 샀다면
이후 수익이 어땠나"를 보여준다. 문제는 일별 신호가 강하게 자기상관된다는 것 —
저평가 신호는 연속된 수백 거래일에 걸쳐 켜지고, 이웃한 날들의 "12개월 후 수익"은
거의 같은 기간을 공유한다. 소박하게 신호일 전부를 표본으로 쓰면 한 번의 랠리를
수십 번 중복 계산해 평균수익과 승률이 과대평가된다.

## 결정

1. **정상 배수 = 직전 1.5년(378거래일) 롤링 중앙값.** 그 시점까지의 데이터만 사용해
   룩어헤드를 방지하고, 일시적 이익 급변(EPS 왜곡)에 강건하도록 평균이 아닌 중앙값을 쓴다.
2. **신호 = 괴리율(정상배수 기반 적정가/주가 − 1) ≥ +30%.** 고정 임계값으로 해석을 단순하게 유지한다.
3. **이벤트 통계(평균수익·승률)는 12개월씩 겹치지 않게 고른 비중복 표본으로만 계산한다.**
   표본이 5개 미만이면 신뢰도 경고를 띄운다.
4. **도구 유효성 지표는 스피어만 순위상관**(저평가율 vs 12개월 후 수익) — 값의 크기가 아닌
   순위로 계산해 이상치와 비선형 관계에 강건하다.
5. **예시 전략 곡선은 신호 다음날 진입**으로 계산해 룩어헤드를 방지한다.
6. 산점도는 전체 관측일을 그대로 보여주되(패턴 관찰용), 수치 통계는 항상 비중복 표본을 쓴다.

## 근거

- 겹치는 보유기간의 수익률 표본은 독립이 아니다. 이벤트 스터디에서 기간을 겹치지 않게
  솎아내는 것은 표본 수를 줄이는 대신 통계를 보수적으로 만드는 표준적 처리다.
- 롤링 "정상 배수"에 중앙값을 쓰면 일회성 손익으로 EPS가 튀는 해에도 기준선이 덜 흔들린다.
- 순위상관은 "더 싸게 산 날일수록 이후 수익이 좋았나"라는 질문을 척도에 무관하게 답한다.
  양수면 이 종목에서 저평가 신호가 유효했고, 0 근처면 무의미, 음수면 밸류트랩 경향이다.

## 검토한 대안

- **겹침 허용 + Newey-West 표준오차 보정**: 통계적으로 더 정교하지만 구현·설명 복잡도가
  학습 도구의 취지를 넘는다. 비중복화가 더 단순하고 보수적이다.
- **피어슨 상관**: 극단값(급등락일)에 민감해 순위상관으로 대체.
- **고정 임계값 대신 분위수 신호**(예: 저평가율 상위 20% 날): 종목 간 비교에는 좋으나
  "적정가 대비 +30%"라는 직관적 해석을 잃는다.

## 부록 — 결과 화면 해석 가이드

| 용어 | 뜻 |
|---|---|
| 저평가 신호 N거래일 | 괴리율 +30% 이상이었던 날의 총수 |
| 비중복 12M 표본 | 12개월씩 겹치지 않게 고른 독립 표본 수. 5개 미만이면 통계 신뢰 불가 |
| 신호 후 12M 평균 | 독립 표본들의 1년 수익 평균. "전체 표본 평균"(아무 날 매수)보다 높아야 신호가 유의미 |
| 그때 플러스 확률 | 표본 중 1년 뒤 수익이 +였던 비율(승률) |
| 저평가↔수익 상관 | 스피어만 순위상관. +면 도구가 이 종목에서 작동, −면 밸류트랩 신호 |

산점도 읽는 법: 점 하나 = 과거 거래일 하루. 가로축 = 그날의 저평가율(오른쪽일수록 모형 기준
싸게 거래), 세로축 = 그날 사서 12개월 보유한 실제 수익. 음영 = 신호 구간(+30%↑),
추세선 우상향 = 저평가가 심할수록 이후 수익이 좋았다는 뜻. 단, 이웃한 날짜의 점들은
거의 같은 정보라 뭉쳐 다닌다 — 점 300개가 독립 증거 300개는 아니다.

## 한계와 재검토 조건

- 단일 종목의 과거 패턴이며 미래를 보장하지 않는다. 화면에도 "참고용 통계"로 명시한다.
- 연간 재무의 일별 보간은 근사이며, 공시 시차를 완전히 반영하지 못한다.
- 표본이 만성적으로 5개 미만인 종목이 많으면 임계값 조정 UI(현재 +30% 고정) 도입을 재검토한다.
