# ADR-0001: 적정가 종합은 동일가중 산술평균으로 한다

- 상태: **대체됨 — [ADR-0003](0003-fair-value-weighted-average.md)이 대체**(2026-07-19 컨센서스 도입으로 4방법 가중평균 전환)
- 날짜: 2026-07-15
- 관련 코드: `src/analysis/valuation.py`의 `evaluate()` 종합 판정 블록

> ⚠️ 이 결정은 더 이상 유효하지 않습니다. 3방법 동일가중 산술평균은 ④ 선행 이익(컨센서스) 방법
> 도입과 함께 4방법 가중평균으로 바뀌었습니다. 현재 설계·근거는 ADR-0003을 보세요.
> 아래 본문은 당시 판단의 기록으로 보존합니다(ADR 규칙: 채택된 문서는 수정하지 않고 대체 표시).

## 맥락

주식 가치평가는 세 가지 방법으로 적정주가를 추정한다 — ① 업종 상대가치(피어 멀티플),
② 역사적 밴드(자기 과거 배수 분포), ③ 수익가치(RIM). 세 방법은 서로 다른 값을 내며,
데이터 품질에 따라 사용 가능한 방법이 1~3개로 달라진다(적자 기업은 RIM 제외,
상장기간이 짧으면 밴드 제외 등). 화면의 판정 막대와 괴리율 계산에는 하나의 대표값이 필요하다.

## 결정

살아남은 방법들의 중간값(mid)을 **동일가중 산술평균**해 `fair_mid`로 쓴다.
low/high도 각각 평균한다. 동시에:

- 신뢰할 수 없는 방법은 평균에 넣지 않고 **사전 제외**한다
  (자사주 매입 등으로 장부자본이 왜곡된 기업의 RIM, 표본 부족한 밴드 등).
- 방법 간 편차(표준편차/평균)로 **신뢰도**를 표기한다 — 15% 미만 높음, 35% 미만 중간, 이상 낮음.
- 세 방법의 개별 결과를 화면에 항상 병기해 평균의 성분을 공개한다.

## 근거

1. **예측 결합(forecast combination) 문헌**: 여러 추정치를 결합하면 개별 추정치보다
   오차가 줄어들고, "최적" 가중치를 데이터로 추정하려는 시도는 가중치 자체의 추정오차
   때문에 동일가중보다 못한 경우가 많다는 결과가 반복 보고되어 있다
   (Clemen 1989 리뷰; 포트폴리오의 1/N 논증과 같은 정신).
2. **업계 관행과의 관계**: 증권사 리서치도 상대가치·절대가치를 병기하고 범위(풋볼필드)로
   제시한다. 종합치는 애널리스트가 업종 특성에 따라 재량 가중하는 경우가 많으나,
   본 도구는 자동화된 학습 도구로서 **주관을 배제한 중립 기본값**이 더 적절하다고 판단했다.
3. **투명성 원칙**: 평균이라는 단순한 규칙일수록 사용자가 검증 가능하다.
   프로젝트의 우선 기준(검증 가능성·정직한 불확실성)과 일치한다.

## 검토한 대안

- **3방법 중앙값**: 튀는 방법 하나에 덜 끌려가는 장점(강건성). 방법이 2개만 남으면
  평균과 동일해 이득이 제한적이고, "가운데 방법 하나가 대표"가 되는 것이 세 관점의
  종합이라는 제품 서사와 어긋나 보류. 신뢰도 '낮음'이 빈발하면 재검토(아래 참조).
- **편차 역가중(정밀도 가중)**: 가중치 추정이 새로운 추정오차를 만든다. 근거 1과 충돌.
- **업종 조건부 가중**(예: 금융업 = RIM/PBR 중심): 이론적으로 타당하나 규칙 유지보수
  부담과 결측 시 동작 복잡성이 커서 자동화 도구의 기본값으로는 과하다.

## 한계와 재검토 조건

- 산술평균은 이상치 방법에 민감하다. 예: 삼성전자(2026-07)에서 세 방법이
  179천/292천/92천으로 갈리자 평균 187천이 RIM에 끌려 내려갔고 신뢰도 '낮음'이 표기됐다.
  이 표기가 정직한 처리이지만, **신뢰도 '낮음'이 빈발하는 종목군이 확인되면
  중앙값 옵션(또는 UI 토글) 도입을 재검토**한다.
- UI에는 계속 "3개 방법 평균"임을 명시해 단일 정답으로 오독되지 않게 유지한다.
